# SideBarny LLM - опис для маркетплейсу (UA)

## Короткий опис (до 132 символів)

Бічна панель Chrome для роботи з LLM: ChatGPT, Claude, Gemini та інші + швидке копіювання тексту/HTML зі сторінок.

## Повний опис

**SideBarny LLM** - це розширення для Chrome, яке відкриває популярні AI-чати у бічній панелі браузера та пришвидшує роботу з контентом сторінок.

### Що вміє розширення

- Працює в бічній панелі (без постійного перемикання вкладок)
- Підтримує кілька LLM-провайдерів в одному інтерфейсі
- Має вбудований вибір елемента для швидкого захоплення контексту
- Копіює видимий текст з обраного елемента
- Копіює HTML-розмітку обраного елемента
- Автоматично кладе результат у буфер обміну
- Дозволяє відкрити провайдера у звичайній вкладці, якщо iframe-режим обмежений на стороні сервісу

### Для кого

- QA / Test Automation Engineers
- Developers
- Product та Support команди
- Усі, хто регулярно переносить контекст зі сторінок у LLM

### Приватність і дозволи

Розширення використовує доступи, потрібні для роботи бічної панелі і захоплення елементів:

- `sidePanel`
- `declarativeNetRequest`
- `scripting`
- `tabs`
- `activeTab`
- `clipboardWrite`
- `host_permissions: <all_urls>`

Дані користувача не відправляються на окремий бекенд розширення. Захоплений контент копіюється у ваш буфер обміну для подальшого використання.
