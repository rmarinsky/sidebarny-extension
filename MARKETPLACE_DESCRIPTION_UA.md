# SideBarny LLM - опис для маркетплейсу (UA)

## Короткий опис (до 132 символів)

Бічна панель Chrome для роботи з LLM: ChatGPT, Claude, Gemini та інші + автовставка тексту/HTML зі сторінок прямо в чат.

## Повний опис

**SideBarny LLM** — це розширення для Chrome, яке відкриває популярні AI-чати у бічній панелі браузера та автоматично вставляє захоплений контент зі сторінок прямо в поле вводу чату.

### Що вміє розширення

- Працює в бічній панелі (без постійного перемикання вкладок)
- Підтримує 8 LLM-провайдерів в одному інтерфейсі
- Вибір елемента у стилі DevTools з підсвіткою при наведенні
- Захоплює видимий текст або HTML-розмітку обраного елемента
- **Автоматично вставляє контент прямо в чат** кожного провайдера — не потрібно Ctrl+V
- Витягує текст з ARIA-атрибутів для елементів без видимого тексту (кнопки, іконки тощо)
- Фолбек на буфер обміну, якщо автовставка неможлива
- Дозволяє відкрити провайдера у звичайній вкладці, якщо iframe-режим обмежений
- Адаптивна тема інтерфейсу під кожного провайдера
- Мінімальні дозволи — розширені доступи запитуються лише при першому використанні

### Приватність і дозволи

Розширення запитує мінімально необхідні дозволи:

- `sidePanel` — бічна панель
- `declarativeNetRequest` — зняття iframe-обмежень для провайдерів
- `storage` — збереження обраного провайдера
- `activeTab` — тимчасовий доступ до активної вкладки

Додаткові дозволи (`scripting`, доступ до сторінок) запитуються лише при першому захопленні контенту.

Дані користувача не відправляються на окремий бекенд розширення. Захоплений контент обробляється локально та вставляється в чат або копіюється у буфер обміну.
